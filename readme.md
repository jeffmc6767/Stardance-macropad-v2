## Bill of Materials (BOM)
I also have a bom.csv 

- 1x XIAO RP2040
- 9x Cherry MX switches
- 9x DSA Keycaps
- 9x 1N4148 DO-35 Diodes.
- 10x WS2812B LEDs
- 2x EC11 Rotary Encoder
- 1x case(I have a 3d printer so I dont actually need funding for this)
- 4x M3x5mx4mm heatset inserts
- 4x M3x16mm screws

## Folder Navigation

* `/xiaopad` — Core QMK firmware files (`info.json`, `config.h`, `keymap.c`, `rules.mk`).
* `/kicad` — Main electronic design schemas and layout assets.
* `/Plotting for PCB` — Complete manufacturing Gerber production packages (zipped).
